# 機能概要

本アプリケーションは、`todo.schema.json` に記載されたデータ形式を用いてToDoを管理するアプリです。

## 提供する主な画面

少なくとも以下の3つの画面を提供します。

- **ToDoの新規作成画面**
  - タイトル(必須)、詳細な説明、ステータス、優先度、期限、タグの入力が可能です。
- **ToDoの編集画面**
  - 既存のToDoの各項目（タイトル、説明、ステータス、優先度、期限、タグ）の更新が可能です。
- **ToDoの一覧表示画面**
  - 登録されたタスクを優先度（高・中・低）や期日順にソート・フィルタリングして表示します。
  - 各タスクの現在のステータス（todo, in_progress, done）も一目で確認できるようにします。

## 主な機能仕様

- **タスクのステータス管理:** `todo`（未着手）, `in_progress`（進行中）, `done`（完了）の3段階で管理します。
- **優先度設定:** `low`（低）, `medium`（中）, `high`（高）の3段階で設定可能です。
- **日付管理:** 作成日時、更新日時、および期日（期限）をISO 8601形式で記録・管理します。
- **タグ付け:** 任意のタグを複数付与し、用途に応じた分類が可能です。
